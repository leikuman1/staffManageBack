# staffManageBack

后端项目定位：基于 **JDK17** 的 **Spring Boot** 应用，使用 **MySQL** 作为主数据存储、**Redis** 做缓存/会话/验证码等加速和解耦，围绕学校人员信息管理提供接口服务。

核心功能模块（六个）：
1. **认证与权限**：账户、角色、菜单/接口权限，登录态存入 Redis。
2. **组织架构**：学院/部门/岗位层级管理，支持上下级关系及岗位编制。
3. **教职工档案**：教职工基础信息、任教信息、资质证书、调动记录等。
4. **学生档案**：学生基本信息、班级与导师关系、学籍/状态流转记录。
5. **考勤与请假**：签到、缺勤、请假流程及审批记录，配合 Redis 做限流与队列。
6. **薪酬与报表**：薪酬/绩效记录、发放记录、结算与对账统计报表导出。

数据库：MySQL 持久化核心业务数据；Redis 用于热点数据缓存、分布式锁、会话/验证码存储和异步消息队列（如考勤事件）。
